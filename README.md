# A personal focus assistant that simulates an Air Traffic Control (ATC) tower to help you disconnect from the computer and avoid hyperfocus.

# Why does this project exist?

For many autistic and neurodivergent individuals, hyperfocus is a common experience. While it can be a superpower for productivity, it can also lead to burnout and neglecting basic needs. Generic "take a break" alerts are often jarring, ineffective, and easy to dismiss. Focus Helper offers a novel solution by transforming these interruptions into an engaging, immersive experience that respects the user's special interests.

By simulating an Air Traffic Control (ATC) tower, it uses a creative, aviation-themed approach to deliver critical alerts. Instead of a simple notification, users receive a "radio call" with urgent instructions, dynamically generated by the Llama 3.2 AI, making the process of disconnecting more effective and enjoyable.
‚ú® Key Features

Air Traffic Control (ATC) Simulation: Aviation-themed audio messages, generated in real-time by Llama 3.2 running locally via Ollama.

Progressive Hyperfocus Alerts: The system has multiple alert levels. The "emergency" intensifies as your continuous usage time increases, from a subtle pop-up to a critical radio call.

Realistic Voice and Radio Effects: Uses the Piper TTS engine for a high-quality local voice, processed with SoX to add filter and compression effects, and mixed with static noise for complete immersion.

Smart Home Integration: Can trigger webhooks for Home Assistant, allowing you to turn on warning lights, play alerts on Alexa, or create any other automation you can think of.

Configurable Wellbeing Questions: Asks dynamic questions about posture, hydration, and wellbeing, with different interaction types (Yes/No, Free Text).

Highly Configurable: Almost everything can be adjusted: alert timings, sounds, AI prompts, volumes, and the actions for each emergency level.

üõ†Ô∏è How It Works

The application runs silently in the background, following a cycle:

Monitoring: Continuously checks mouse activity to calculate your uninterrupted usage time.

Detection: Compares your usage time with the configured thresholds for each alert level.

Execution: When a threshold is reached, the system triggers a sequence of actions defined for that level. This can include:

Showing a pop-up or desktop notification.

Playing a specific alert sound.

Generating text with Llama 3.2, synthesizing the voice with Piper, applying radio effects, and playing the final message.

Sending a webhook to Home Assistant.
